# Praktikum Otomata I dan II

> [!TIP]
> Make sure you have python atleast 3.10 installed on your system <br>
> Follow the steps before running the python files on `Prerequisite`

### Prerequisite
1. Python Interpreter installed (atleast 3.10)
2. Tkinter (Should be installed automatically on Windows)

### How to run
1. All the code exists on `src/` folder
2. Run using python

```
# Run tokenizer program (Praktikum P01 - Minggu II)
python P01_tokenizer.py

# Run FSM program (Praktikum P02 - Minggu III)
python P02_fsm.py
```
